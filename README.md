# if you want to run this project, take this steps

*firstly open cmd
*pip install virtualenv
*cd dektop --> cd Python-Django-TodoApp-main --> cd Python-Django-TodoApp-main   (if project folder on desktop)
*virtualenv todoenv
*todoenv\Scrips\Activate
*pip install django
*cd todo_project
*python manage.py migrate
*python manage.py makemigrations
*python manage.py createsuperuser
*python manage.py migrate
*python manage.py runserver
